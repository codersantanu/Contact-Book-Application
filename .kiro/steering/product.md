# Product Overview

Contact Book is a web-based contact management application that allows users to store, search, and manage their contacts in a centralized database.

## Core Features

- Add, edit, and delete contacts with name, phone, and optional email
- Real-time client-side search across all contact fields
- Modal-based UI for adding/editing contacts
- Flash messaging for user feedback
- Responsive grid layout for contact cards

## User Experience

The application provides a clean, modern interface with a teal/turquoise color scheme. All CRUD operations are performed through an intuitive modal dialog, and contacts are displayed as cards with emoji icons for visual clarity.
