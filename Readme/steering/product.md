# App Overview

Contact Book is a web-based contact management application that allows users to store, search, and manage their contacts in one centralized location.

## Core Features

- Add, edit, and delete contacts
- Store contact information: name, phone number, and email (optional)
- Real-time client-side search by name, phone, or email
- Modal-based forms for adding and editing contacts
- Flash messages for user feedback on operations
- Responsive grid layout for contact cards

## User Experience

The application provides a clean, modern interface with:
- Gradient header with branding
- Card-based contact display
- Inline search functionality
- Confirmation dialogs for destructive actions
